CSS 实现隐藏滚动条同时滚动

发表于 2018-07-25 | 阅读次数:

网站中部分页面为了更好的用户体验，是否可以隐藏滚动条，同时又保证页面可以滚动？

使用 overflow:hidden 隐藏滚动条，但存在的问题是：页面或元素失去了滚动的特性。
由于只需要兼容移动浏览器（Chrome 和 Safari），于是想到了自定义滚动条的伪对象选择器
::-webkit-scrollbar。

关于这个选择器的介绍可以参考：

Styling Scrollbars

Custom Scrollbars in WebKit

应用如下 CSS 可以隐藏滚动条：

.element::-webkit-scrollbar {display:none}

如果要兼容 PC 其他浏览器（IE、Firefox 等），国外一位才人 John Kurlak 也研究出了一种办法。在容器外面再嵌套一层overflow:hidden内部内容再限制尺寸和外部嵌套层一样，就变相隐藏了。

参考：Hiding Vertical Scrollbars with Pure CSS in Chrome, IE (6+), Firefox, Opera, and Safari
